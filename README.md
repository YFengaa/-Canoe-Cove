# -Canoe-Cove
轻舟树洞（Canoe Cove）是一款面向中学生的情绪倾诉与考前减压应用，参加TRAE AI创造力大赛青少年身心健康赛道。项目以"轻舟已过万重山"为意象，为面临学业压力、人际困扰和成长迷茫的青少年提供安全、温暖、纯匿名的情绪出口。应用包含情绪树洞对话、同龄人匿名墙、考前减压工具包、治愈明信片、情绪日历五大功能模块。采用单文件HTML架构，浏览器直接打开即用，无需安装部署。内置AES-256-GCM端到端加密与Supabase云端同步，确保用户数据隐私安全。AI能力接入豆包大模型，构建多级fallback架构，保证离线可用。危机干预机制检测关键词并联动全国心理援助热线400-161-9995，守护青少年安全底线。
